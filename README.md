# ChangeIS Explainer Videos

## Overview

This repository contains explainer videos, audio, and supporting assets
for ChangeIS K.K. and the GuruSuite product family.

ChangeIS K.K. is a Tokyo-based consulting and product firm
specializing in AI, security, monitoring, and operational tooling
for organizations that need to move fast and get it right.

The purpose of this repository is to provide:

- explainer videos for each ChangeIS product
- companion audio tracks for each video
- supporting documentation and scripts
- brand and logo assets

The videos follow a consistent structure:
short, specific, and hope-first —
showing the better world each product enables.

# GuruSuite Products

GuruSuite is the ChangeIS product family.
Each product delivers a focused area of organizational wisdom
that learns and grows with the teams that use it.

| Product | Focus | Tagline |
|------|------|------|
| Maestro | Testing | Testing wisdom that grows with you |
| Weavr | Knowledge | Knowledge wisdom that grows with you |
| Pulse | Monitoring | Monitoring wisdom that grows with you |
| Vigil | Events | Event wisdom that grows with you |
| Relay | Delivery | Delivery wisdom that grows with you |
| Galen | Advisory | Advisory wisdom that grows with you |
| Nexa | DevOps | DevOps wisdom that grows with you |
| Haven | Security | Security wisdom that grows with you |
| Fluid | Collaboration | Collaboration wisdom that grows with you |

Each explainer video follows a consistent three-beat structure:

| Beat | Purpose |
|------|------|
| 1 — Imagine | Open with the better world the product enables |
| 2 — Here is how | The product and its living library |
| 3 — Call to action | Product name, tagline, GuruSuite / ChangeIS lockup |

# Design Principles

- Hope-first — lead with the better world, not the problem
- Specific — each video speaks to a real audience and a real situation
- Short — approximately 25 seconds per product
- Zero FUD — no fear, uncertainty, or doubt
